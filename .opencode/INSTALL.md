# Quincy Claw — OpenCode Installation

```bash
git clone https://github.com/NicoAcosta/quincy-claw.git ~/.opencode/plugins/quincy-claw
```

Then add to your OpenCode config:

```json
{
  "plugins": ["~/.opencode/plugins/quincy-claw"]
}
```

Skills will be available as `/play`, `/studio`, `/vibe`, and `/quincy`.
